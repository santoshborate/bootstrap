Button Styles

Bootstrap provides different styles of buttons.

To achieve the button styles above, Bootstrap has the following classes:

        .btn
        .btn-default
        .btn-primary
        .btn-success
        .btn-info
        .btn-warning
        .btn-danger
        .btn-link
